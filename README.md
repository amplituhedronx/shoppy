# shoppy

A simple shared shopping list web app.

## Features
- Login with two users (MHU / CBR)
- Shared shopping list for all users
- Add, check, and delete items
- Mobile-friendly dark theme with orange accents
- No backend required (pure frontend + localStorage)

- Username: `CBR` | Password: `cbr123`

## Deployment
This is a static site. Easy to deploy on Render, Vercel, Netlify, or GitHub Pages.

## Local Development
Just open `index.html` in any browser.
